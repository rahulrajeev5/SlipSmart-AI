# SlipSmart AI – UI Design

## Purpose

This document defines the initial user interface structure for the SlipSmart AI MVP.

## Main Pages

### Landing Page

Purpose:

Introduce SlipSmart AI and explain the product.

Sections:

- Hero section
- Product value proposition
- How it works
- Key features
- Call to action

### Dashboard

Purpose:

Show the main user workspace.

Sections:

- Upcoming matches
- Prediction highlights
- Slip generator shortcut
- Recent generated slips

### Matches Page

Purpose:

Display upcoming matches.

Each match card shows:

- League
- Home team
- Away team
- Kickoff time
- Status

### Predictions Page

Purpose:

Display prediction picks for matches.

Each prediction card shows:

- Match
- Market
- Pick
- Probability
- Odds
- Risk level

### Slip Generator Page

Purpose:

Generate diversified slips.

Inputs:

- Budget
- Number of slips
- Picks per slip
- Strategy
- Max overlap

Outputs:

- Generated slips
- Stake per slip
- Total odds
- Potential return
- Average probability
- Risk score
- AI explanation

### AI Assistant Page

Purpose:

Allow users to ask questions about predictions and strategies.

MVP version can use mock responses.

Future version will use OpenAI or Gemini.

## Design Principles

- Mobile first
- Simple and clean interface
- Dark sports-tech theme
- Clear risk labels
- Avoid overwhelming users with too much data
- Explain predictions in plain language
